# vscode-text-utils

This is a Visual Studio Code extension that provides utilities for text manipulation.

## Features

- **Base64**
  - Encode/Decode
- **JSON**
  - Minify
  - Encode/Decode
- **Hashing**
  - MD5
  - SHA-1
  - SHA-256
- **URI**
  - Encode/Decode
- **UUID**
  - Generate v4 UUIDs
